# 🧠 Quizzerd — AI-Powered Quiz Application

**Quizzerd** is an intelligent, AI-driven web application that generates dynamic multiple-choice quizzes on any topic using the Groq AI API. Built as a university project, it features a complete CRUD architecture, user authentication, a global leaderboard, community exploration, and question bookmarking.

![Quizzerd Banner](https://via.placeholder.com/1000x400?text=Quizzerd+-+AI+Quiz+Platform) *(You can replace this link with a real screenshot of your dashboard)*

## ✨ Key Features

*   **🤖 AI Quiz Generation:** Instantly generate factual, challenging MCQs on any subject using Groq Cloud API (Llama 3 model).
*   **🔐 Authentication:** Secure user registration and login system using JWT (JSON Web Tokens).
*   **📊 Personalized Dashboard:** Track total score, accuracy, and quizzes played.
*   **🌍 Explore Community:** Browse, search, and filter quizzes generated by other users. Play trending quizzes!
*   **🏆 Global Leaderboard:** Compete with other users and climb the ranks to the top 3 podium.
*   **🔖 Bookmark System:** Save specific questions and correct answers for future review.
*   **⚙️ Full CRUD Operations:** 
    *   **Create:** Generate new quizzes.
    *   **Read:** Play quizzes and review detailed results.
    *   **Update:** Edit the quiz metadata (Subject & Difficulty).
    *   **Delete:** Remove unwanted quizzes permanently.

## 🛠️ Tech Stack

*   **Frontend:** HTML5, CSS3, Vanilla JavaScript (DOM Manipulation)
*   **Backend:** Node.js, Express.js
*   **Database:** MySQL (Relational Database)
*   **AI Integration:** Groq API SDK
*   **Security:** bcryptjs (Password Hashing), jsonwebtoken (Auth)
